How to get the cookie, sessionId, candidateId
  1. Log in on the site and start a booking search until you reach the date/city page.
  2. Open DevTools, Network tab, enable Preserve log.
  3. Find the GetAvailableCBTCentersBySessionId request.
  4. Headers tab, Request Headers, copy the value after Cookie:.
  5. Payload tab, copy sessionId and candidateId.
These expire. If checks start failing, repeat this for fresh values.
